# NewsBot Intelligence System 2.0 (ITAI 2373 Final Project)

## 1. Overview

NewsBot 2.0 is an end-to-end NLP system that ingests news articles and provides:
- Multi-class news topic classification
- Topic modeling (LDA + NMF)
- Sentiment and emotion insights
- Named Entity Recognition (NER)
- Multilingual detection, translation, and cross-lingual analysis
- A conversational interface that lets users query the system in natural language

This repository extends the midterm version of NewsBot into a modular, production-style codebase.
